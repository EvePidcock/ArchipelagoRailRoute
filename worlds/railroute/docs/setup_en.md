# Rail Route Archipelago Setup Guide

## Required Software

Host:

- [Archipelago](https://github.com/ArchipelagoMW/Archipelago/releases/latest)
- [The Rail Route apworld](github release link)

Player:

- [The Rail Route client mod](github release link)

## How to play

First, you need a room to connect to. For this, you or someone you know has to generate a game.  
This will not be explained here,
but you can check the [Archipelago Setup Guide](/tutorial/Archipelago/setup_en#generating-a-game).

Extract the content of the client mod into `\AppData\LocalLow\bitrich\Rail Route\mods` (you may have to create a mods folder if one does not exist).
